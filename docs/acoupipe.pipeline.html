pipeline¶

All classes in this module can be used to calculate and provide data (via feature extraction) and to control the random sampling (via BaseSampler derived classes).

DataGenerator

Abstract base class that serves as a data generator.

BasePipeline(*args, **kwargs)

Class to control the random process and iteratively extract and pass a specified amount of data.

DistributedPipeline(*args, **kwargs)

Class to calculate data (extract features) in parallel to build large datasets.
