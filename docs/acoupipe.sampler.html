All classes in this module are random processes that meant to be used to manipulate instances i.e. their attribute values according to a specified random distribution (random variable).

BaseSampler

Base class that represents a random process manipulating attributes of an instance or a list of instances according to a specified random distribution.

NumericAttributeSampler

Random process that manipulates attributes of numeric type (e.g.

SetSampler

Random process that draws one or multiple values from a given set of values and assigns those to the target.

SourceSetSampler

From a given set of sources (type acoular.SamplesGenerator), SourceSetSampler draws one or multiple sources from this set and assigns those to one or more SourceMixer instances.

ContainerSampler

Special case of a Sampler to enable the use of an arbitrary function performing the sampling and that can be passed to a BasePipeline derived class.

PointSourceSampler

Random process that samples the locations of one or more instances of type PointSource.

MicGeomSampler

Random process that samples the microphone positions of one instance of type acoular.MicGeom.
