The following modules are part of AcouPipe:

sampler

All classes in this module are random processes that meant to be used to manipulate instances i.e. their attribute values according to a specified random distribution (random variable).

pipeline

All classes in this module can be used to calculate and provide data (via feature extraction) and to control the random sampling (via BaseSampler derived classes).

writer

Provides classes to store the data extracted by BasePipeline derived classes.

loader

Provides classes to load the datasets stored with BaseWriteDataset derived classes.
