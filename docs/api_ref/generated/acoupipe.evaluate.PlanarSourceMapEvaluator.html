PlanarSourceMapEvaluator¶

class acoupipe.evaluate.PlanarSourceMapEvaluator¶

Bases: acoupipe.evaluate.BaseEvaluator

Class to evaluate the performance of microphone array methods on planar grid-based source maps.

This class can be used to calculate different performance metrics to assess the performance of a source mapping method, including:

  • specific level error (Herold and Sarradj, 2017)

  • overall level error (Herold and Sarradj, 2017)

  • inverse level error (Herold and Sarradj, 2017)

sourcemap = CArray( …¶

an array of shape=(nf,nx,ny) containing sourcemaps to evaluate. nf is the number of frequencies, nx is the number of spatial samples in x-direction, ny is the number of spatial samples in y-direction.

grid = Instance(RectGrid, …¶

Grid-derived object that provides the grid locations.
