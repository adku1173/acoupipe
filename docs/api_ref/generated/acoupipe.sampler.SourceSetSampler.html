SourceSetSampler¶

class acoupipe.sampler.SourceSetSampler¶

Bases: acoupipe.sampler.SetSampler

From a given set of sources (type acoular.SamplesGenerator), SourceSetSampler draws one or multiple sources from this set and assigns those to one or more SourceMixer instances.

target = List(Instance(SourceMixer, ()), …¶

a list of acoular.SourceMixer instances

numsamples = Int(1, …¶

number of samples to be drawn from the set

attribute = Enum("sources",  …¶

attribute where drawn sources are assigned to. Fixed to sources attribute of acoular.SourceMixer.

set_value(target, value)¶

assigns the sampled values to the SourceMixer instances; only for internal use

sample()¶

this function utilizes rvs() function to draw values from set list that are going to be assigned to the target instance attribute.
