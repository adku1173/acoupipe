BasePipeline¶

class acoupipe.pipeline.BasePipeline(*args, **kwargs)¶

Bases: acoupipe.pipeline.DataGenerator

Class to control the random process and iteratively extract and pass a specified amount of data.

This class can be used to calculate data (extract features) by assigning a name and a callable function to features. Furthermore this class automatically controles the sampling of instances of type BaseSampler specified to the sampler list. Re-seeding is performed at each iteration if random_seeds are given.

sampler = List([], …¶

a list with instances of BaseSampler derived classes

features = Dict(key_trait=Str(""),  …¶

dictionary consisting of feature names (key) and their extraction functions as callable (value) one can either pass a callable (e.g. features = {“print” : lambda: print(“extract..”)}) or a tuple containing the callable and their arguments (e.g.: features = {“print” : (lambda: print(x), “extract..”)}).

random_seeds = List(range, …¶

a list of range(seeds) associated with sampler objects in sampler. A new seed will be collected from each range object during an evaluation of the get_data() generator. This seed is used to initialize an instance of numpy.random._generator.Generator which is passed to the random_state of the samplers in sampler. If not given, get_data() relies on numsamples.

numsamples = Int(0, …¶

number of data samples to calculate by get_data(). Will be superseded by the random_seeds attribute if specified.

logger = Trait(logging.getLogger(__name__), …¶

logger instance to log calculation times for each data sample

provides the extracted features, sampler seeds and indices.

dict
