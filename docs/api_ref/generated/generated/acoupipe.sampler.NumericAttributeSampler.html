NumericAttributeSampler¶

class acoupipe.sampler.NumericAttributeSampler¶

Bases: acoupipe.sampler.BaseSampler

Random process that manipulates attributes of numeric type (e.g. int, float) according to a specified random distribution.

attribute = Str("",  …¶

attribute of the object in the target list that should be sampled by the random variable

normalize = Bool(False,  …¶

whether to normalize the drawn values (maximum element equals 1). if single_value is set to True, this has no effect.

order = Either("ascending","descending")¶

whether to order the drawn values in ascending or descending order for all objects in the target list. if single_value is set to True, this has no effect. If no value is set (order =None), no ordering is performed.

filter = Callable( …¶

sampled value filter (resample if callable filter returns False)

order_samples(samples)¶

internal function to order drawn values

sample()¶

this function utilizes rvs() function to draw random values from random_var that are going to be assigned to the target instance attribute via internal set_value() method.
