BaseSampler¶

class acoupipe.sampler.BaseSampler¶

Bases: traits.has_traits.HasPrivateTraits

Base class that represents a random process manipulating attributes of an instance or a list of instances according to a specified random distribution.

This class has no functionality and should not be used in practice.

target = Trait(list, …¶

a list of instances which attributes are to be manipulated

random_var = Instance(_distn_infrastructure.rv_frozen, …¶

the random variable specifying the random distribution

random_state = Either(int,RandomState,Generator, …¶

the state of the random variable random_var

single_value = Bool(False, #TODO: better rename to same_sample …¶

manages if the same sampled value is chosen for all objects in the target list if False, one value for each target is drawn

rvs(size=1)¶

random variable sampling (for internal use)

sample()¶

this function utilizes rvs() function to draw random values from random_var (no functionality in this class).
