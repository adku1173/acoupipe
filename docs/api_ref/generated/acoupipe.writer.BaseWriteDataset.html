BaseWriteDataset¶

class acoupipe.writer.BaseWriteDataset¶

Bases: acoupipe.pipeline.DataGenerator

Base class for all derived classes intended to write data from BasePipeline instances to a specific file format. This class has no functionality and should not be used.
