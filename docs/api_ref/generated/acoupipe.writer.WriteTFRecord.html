WriteTFRecord¶

class acoupipe.writer.WriteTFRecord¶

Bases: acoupipe.writer.BaseWriteDataset

Class intended to write data from BasePipeline instances to a .tfrecord file that can be consumed by tensorflows tf.data API. Stores data in binary format.

name = File(filter=['*.tfrecords'],  …¶

Name of the file to be saved.

encoder_funcs = Dict(key_trait=Str(), value_trait=Function(),  …¶

Dictionary with encoding functions (dict values) to convert data yielded by the pipeline to binary format of .tfrecord file. The key values of this dictionary are the feature names specified in the features attribute of the source object.

write_infofile = Bool(True,  …¶

if True, writes an additional .txt file containing the names, types and shapes of the features stored in the tfrecord data set.

options = Trait(None,tf.io.TFRecordOptions)¶

Trait to set specific options to the .tfrecord file.

save()¶

saves the output of the get_data() method of a BasePipeline derived class instance specified at source to .tfrecord file format.

get_data()¶

Python generator that saves the data passed by the source to a *.tfrecord file and yields the data to the next object.
