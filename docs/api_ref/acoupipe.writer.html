Provides classes to store the data extracted by BasePipeline derived classes. Current implementation includes WriteH5Dataset to save data to .h5 files and WriteTFRecord to save .tfrecord files. The latter can be efficiently consumed by the Tensorflow framework for machine learning and is optional.

BaseWriteDataset

Base class for all derived classes intended to write data from BasePipeline instances to a specific file format.

WriteH5Dataset

Class intended to write data to a .h5 file.

WriteTFRecord

Class intended to write data from BasePipeline instances to a .tfrecord file that can be consumed by tensorflows tf.data API.
